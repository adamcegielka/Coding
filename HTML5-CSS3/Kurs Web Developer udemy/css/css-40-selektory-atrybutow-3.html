Lekcja 40 - Selektory atrybutów [attribute*="value"]

*= Wybiera element jesli w atrybucie gdziekolwiek znajduje się dany text,
wielkosć liter ma znaczenie


Wielkie litery, nie bedzie red
box jest red
jest red z foo1234
jest red z _box-foo-1234
nie działa z _box-fo0


<style>
    div[class*="foo"] {
        background-color: red;
        border: 2px solid darkred;
    }
</style>



<div class="FoO">Wielkie litery, nie bedzie red</div>
<div class="klasa1 foo klasa3">box jest red</div>
<div class="foo1234">jest red z foo1234</div>
<div class="_box-foooo-1234">jest red z _box-foo-1234</div>
<div class="_box-fo0">nie działa z _box-fo0</div>
