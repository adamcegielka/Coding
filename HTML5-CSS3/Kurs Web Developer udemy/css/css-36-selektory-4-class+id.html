Selektory class + id

Można odwołać się do danego elementu po id oraz class

.class#id


Komentarz 1: Lorem ipsum dolor sit amet.

Komentarz 2: Lorem ipsum dolor sit amet.



<style>
    #comments {
        background-color: gray;
    }
    .comment {
        font-style: italic;
    }
    .comment#comment-best {
        font-weight: bold;
    }
</style>



<section id="comments" class="container">
    <article id="comment-best" class="comment"><p>Komentarz 1: Lorem ipsum dolor sit amet.</p></article>
    <article class="comment"><p>Komentarz 2: Lorem ipsum dolor sit amet.</p></article>
</section>
