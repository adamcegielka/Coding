    }

    }

    }

    }

    }
</style>

        </article>
    </section>
</article>


Każdy element w <article> jest jego potomkiem
nie każdy jest jego dzieckiem

Dzieci elementu <article>: to bezpośredni potomkowie

  • <header>
  • <p>
  • <footer>
  • <section>

potomkowie elementu <article>:

  • <h1>
  • <p>
  • <article>
