< picture > - Obrazki mogą być responsywne w zależnosci od wielkości ekranu urządzenia:

 < picture >
   < source media=" (min-width:1100px) " srcset="large-image.jpg" >
   < source media=" (min-width:700px) " srcset="medium-image.jpg" >
   < source media=" (min-width:450px) " srcset="small-image.jpg" >
 < /picture >
