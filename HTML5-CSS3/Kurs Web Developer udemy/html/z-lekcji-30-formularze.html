< form action="login.php" method="post" >< /form >

action - atrybut określający adres, gdzie ma być wysłany formularz

method - metoda wysyłki formularzu "get" lub "post", get - domyślnie

    a) post - wysyłanie danych do skrypty w sposób niewidoczny dla użytkownika

    b) get - wysyłanie danych do skryptu poprzez zakodowanie danych URL, domyślna forma

autocomplete - on/off czy przeglądarka może wypełnić pole podpowiedzi

novalidate - formularz nie będzie sprawdzony przed wysyłką

enctype - sposób kodowania tzw MIME-type:

    a) application/x-www-form-urlencoded - domyślnie

    b) multipart/form-date - używany gdy jest pole input type="file" do wysyłki pliku


Dodawanie pliku

application/pdf - dokumenty pdf

image/jpeg - obrazek jpge

text/html - plik html
